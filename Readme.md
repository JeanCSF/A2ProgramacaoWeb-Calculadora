Usando estrutura de OPERADORES crie uma calculadora básica para as 4 operações aritméticas padrão 
(SOMA, SUBTRAÇÃO, MULTIPLICAÇÃO E DIVISÃO)

Calculadora simples feita em PHP como parte de atividade avaliativa A2 para matéria de programação Web

Como usar:

Digitar os números a serem calculador nos 2 inputs que aparecem.

No <select> escolher qual será a operação matématica a ser utilizada.

O resultado aparecera na tela.
